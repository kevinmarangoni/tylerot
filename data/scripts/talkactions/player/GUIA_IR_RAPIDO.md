# 🚀 GUIA RÁPIDO - COMANDO /IR

## 🎯 **COMANDOS PRINCIPAIS**

### **Teleporte:**
```
/ir <local>            # ✅ Teleporta para o local
```

### **Ajuda:**
```
/ir                    # ❓ Ajuda básica
/ir help               # ❓ Ajuda completa
/ir list               # 📋 Lista todos os locais
```
